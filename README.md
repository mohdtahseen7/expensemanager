# expensemanager
## Setup Instructions
1. **Clone the repository**
   git clone <your-repository-url>
   cd expense_manager
2. **Install dependencies**
   flutter pub get
3. **Run the application**
   flutter run
## Database Used
**Cloud Firestore** - A NoSQL cloud database from Firebase
## State Management Approach
**Provider Pattern** - Used for efficient state management across the application
## Features

- User registration and authentication
- Add, view, and filter income records by date
- Add, view, and filter expense records by date with categories
- Dashboard with Today/This Week/This Month summaries
- Comprehensive reports with analytics (highest, lowest, averages)
- Real-time data synchronization
- Pull-to-refresh functionality
- Logout functionality
